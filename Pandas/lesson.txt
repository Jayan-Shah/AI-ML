// Panndas Data Structure:-
1. Series : One dimension labeled array
2. Data Frame : Two dimension labeled array kind of table

Series:-
- pd.Series take care S is capital and its case sensitive
- It will simply convert list into kind of dictonary assign values from array to each index...
// 
s = pd.Series([10,20,30], index=["a","b","c"])

//

 c    30
 dtype: int64

DataFrame:-
//
data = {"Name": ["Alice","Bob"],"Age": [25,30]}
df = pd.DataFrame(data)
//
O/P:---

0  Alice   25
1    Bob   30
